Use this page to upload product data in JSON, CSV, or TXT format. Ensure the file follows the correct format as shown in the examples below.


Example JSON Format

[

    "productId": 1001,
    "name": "Citizen Marvel Avengers Men's Watch",
    "description": "A stylish watch for everyday use.",
    "category": "Unisex",
    "imagePath": "img/citizen-aw2085.jpeg",
    "price": 2545.00
  },
  {
    "productId": 1002,
    "name": "Citizen Men's Watch Drive",
    "description": "A stylish watch for everyday use.",
    "category": "Mens",

  }
]
